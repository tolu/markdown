## Markdown tricks
### Collapsible content with `<details>`

[`<details>` on MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/details)

Example:  
```html
<details>
  <summary>This is the header</summary>
  And here is the expandable content!
</details>
```
